# Gerenciamento-Financeiro

Esta ferramenta foi desenvolvida para máquinas com sistemas operacionais (OS) Windows, portanto, caso a máquina tenha outra OS deverá realizar algumas alterações no código fonte.

Apos alerações realizadas basta abrir o terminal no direotório raiz deste projeto e executar o seguinte comando:

```
cd <diretorio da projeto>
```
```
g++ src/main.cpp src/Transaction.cpp src/FinanceManager.cpp -o <nome do executavel>
```
```
./<nome do executavel> 
```

Caso não queira realizar nenhuma alteracao basta executar os seguintes comandos

```
cd <diretorio da projeto>
```
```
financeManager.exe 
```
